# Traffic Simulator — Confirmed Intent

Outcome: A single-page web app where a hobbyist sets light timings, driver reaction delay (sampled), max velocities, spacing, accel/brake, and runs one trial to see if the tracked car crosses the stop line before green ends; shows an animated 2D top-down view (metric).

User: Personal experimenter / hobbyist.

Why now: Explore how parameter changes affect pass/fail for a given car.

Success: Run a trial, see animation + clear pass/fail result; default queue = 10 cars, adjustable per trial.

Constraint: Simple kinematic car model, single-trial runs, no batch/CSV/export, metric units.

Not doing: Batch experiments, IDM or advanced models, server-side storage, real-time sliders.
